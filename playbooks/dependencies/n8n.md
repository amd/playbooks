# n8n

<!-- @os:windows -->

1. Open PowerShell or Command Prompt
2. Install n8n globally using npm:
```cmd
npm install -g n8n
```

<!-- @os:end -->

<!-- @os:linux -->

Install n8n globally using npm:
```bash
npm install -g n8n
```

> **Note**: See [n8n Documentation](https://docs.n8n.io/hosting/installation/npm/) for more installation options.

<!-- @os:end -->
