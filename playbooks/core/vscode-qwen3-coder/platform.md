# Platform Configuration

This document describes the expected platform configurations for running this playbook.

## Windows

### LMStudio Installation

LMStudio should be pre-installed using the AMD Portable Package

| Component | Version | Location |
|-----------|---------|----------|
| **lmstudio_windows_portable_amd.7z** | 0.3.37 | `C:\ProgramData\lmstudio` |

### VSCode Installation

VSCode should be pre-installed using the AMD Portable Package

| Component | Version | Location |
|-----------|---------|----------|
| **vscode_windows_portable_amd.7z** | 1.108.1 | `C:\Program Files\Microsoft VS Code\` |

### Required Models

The following models must be present in the LMStudio models directory (`C:\.lmstudio\models\`):

| Model Type | Filename | Size | Location | Download |
|------------|----------|------|----------|----------|
| Language Model | `Qwen3-Coder-30B-A3B-Instruct-Q4_0.gguf` | 7.49 GB | `models/Qwen3-Coder-30B-A3B-Instruct-GGUF/` | [Link](https://huggingface.co/unsloth/Qwen3-Coder-30B-A3B-Instruct-GGUF/blob/main/Qwen3-Coder-30B-A3B-Instruct-Q4_0.gguf) |
